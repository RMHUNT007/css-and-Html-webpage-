HTML&CSS Basics Summery

HTML Summery

HTML (Hyper Text Markup Language) is used to define our page content,structure and meaning. You don't use it for styling purpose.Use CSS for that instead

  • HTML uses " elements" to describe (annote) content
  • HTML elements typically have an opening tag,content and then a closing tag
  • You can also have void (empty) elements like images
  • You can also configure elements with attributes
  • There's a long list available elements but you'll gain experience over line, no worries.

Learn more about all available HTML elements on the MDN HTML elements references

CSS Summery

CSS (Cascading Style Sheets) is used for styling your page content.

  • Styles are assigned via property-value pairs
  • You can assign styles via the 'styles' attribute
  • To avoid code duplication, you typically use global styles (e.g. via the'syyles' element) instead
  • Alternatively,you can work with element stylesheet files which you "link" to
  • When working with CSS,concepts like "Intheritance","specifity and the "box model" should be understood

Learn more about all available CSS properties and values on the MDN CSS property reference
